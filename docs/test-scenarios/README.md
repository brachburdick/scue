# Cross-Feature Test Scenario Matrices

This directory contains test scenario matrices for concerns that span multiple features.

**When to add a file here vs. `specs/feat-[name]/test-scenarios.md`:**
- Feature-specific scenarios → `specs/feat-[name]/test-scenarios.md`
- Cross-feature scenarios (e.g., bridge lifecycle, network resilience) → here, as `[area].md`

**Template:** `templates/test-scenarios.md`
**Maintained by:** QA Tester (updates actual/status fields), Architect (adds new scenarios during spec phase)
